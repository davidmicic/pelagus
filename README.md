# Task manager

## How to run

### To build and run the application, follow next steps:
+ navigate to ```/pelagus``` folder 
+ execute the following command: ```make compose-run```

### In case if make is not installed, follow next steps:
+ navigate to ```/pelagus``` folder 
+ execute commands: 
    - docker-compose build
    - docker-compose up
